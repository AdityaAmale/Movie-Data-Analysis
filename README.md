Movie Analytics Dashboard | Power BI

This project presents an interactive Movie Analytics Dashboard built using Power BI, designed to analyze movie industry trends across multiple dimensions such as budget, gross revenue, genres, ratings, countries, and time.

The dashboard focuses on delivering clear insights with a modern UI, enabling users to explore financial performance and popularity patterns of movies efficiently.
